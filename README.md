# EduStream - Decentralized Educational Content Access Platform

A blockchain-based platform that enables students to access up-to-date educational content on demand with verified access control.

## Features

- Decentralized content registration with IPFS integration
- Blockchain-verified access control for students
- Creator-managed content permissions
- Transparent content catalog

## Smart Contract Functions

### Public Functions

- `add-content` - Register new educational content
- `grant-access` - Grant student access to specific content

### Read-Only Functions

- `get-content` - Retrieve content details
- `check-access` - Verify student access permissions
- `get-content-count` - Get total number of registered contents

## Usage

Content creators can register educational materials and grant access to students. All access records are permanently stored on the Stacks blockchain.

## Technology Stack

- Stacks Blockchain
- Clarity Smart Contracts
- IPFS for content storage
